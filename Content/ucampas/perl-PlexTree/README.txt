
Perl-PlexTree -- a simple PlexTree library
------------------------------------------

Markus Kuhn -- http://www.cl.cam.ac.uk/~mgk25/


PlexTrees are a flexible hierarchical general-purpose data model. The
perl-PlexTree package is a simple and easy to use library for working
with them.

For details, please read the PlexTree.3 man page.

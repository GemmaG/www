Infrastructure

Updating the OCaml Labs web pages

The pages are version-controlled in Github at http://github.com/ocamllabs/www. You will need a Github account, and be added to the ocamllabs organisation. Contact Anil Madhavapeddy to get this done.

All pages are formatted via the ucampas tool, so all files have a *-b.html filename. If you add a new file or subdirectory, be sure to edit sync.sh and the uconfig.txt to add the entries there.

The repository has a local copy of ucampas so that you can build it on your laptop. Once you push to the remote Github repository, it will be synched to the live website via a cronjob within the hour.

OCaml for Raspberry Pi (partly) works straight out of the box on Raspbian.

Unfortunately there are (as yet) no pre-compiled ARMv6 binaries suitable to download so it's currently necessary to download the source tree and build it yourself.

Just download the latest version (you need the "source tarball" ocaml-4.00.0.tar.gz ) from http://caml.inria.fr/ocaml/release.en.html,
unzip it with tar zxf ocaml-4.00.0.tar.gz and
read the files in capital letters, such as INSTALL, for (the usual) instructions, which can be abbreviated to: 

$ ./configure
$ make world
$ sudo make install

This (assuming default options) provides "ocaml" (and others) executable binaries in /usr/local/bin which is on the default Raspbian search path.

Also note that the ARM native code OCaml compiler ("make opt") does not yet work (we presume this is ARMv6 and ARMv7 instruction set differences).

The OCaml manuals is available from the above download site too.
